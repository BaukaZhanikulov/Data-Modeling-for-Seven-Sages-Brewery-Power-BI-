Data Modeling for Seven Sages Brewery (Power BI)

В этом проекте я выступал в роли дата-аналитика в компании Seven Sages — это пивоварня, у которой данные были разбросаны по разным источникам. Основная задача была — собрать всё в одну модель данных, навести порядок и построить отчёт, который помогает финансовому директору быстро анализировать продажи и прибыль по продуктам.

Что я делал

Загрузил данные из разных источников (Excel файлы, папки, PDF, валютные курсы)

Объединил, очистил и привёл данные к одной структуре в Power Query

Определил таблицу фактов и таблицы измерений

Построил модель данных с корректными связями (one-to-many)

Создал таблицу дат и учёл финансовый год компании (начинается в октябре)

Написал DAX-меры для:

продаж (USD и CAD)

себестоимости

маржи (gross profit margin)

прибыли по продуктам

Создал двухстраничный Power BI отчёт для анализа:

продаж по клиентам и типам клиентов

валовой прибыли по продуктам
